Disable Microsoft Spy Net

ABOUT
Unenroll from the Microsoft Spy Net

	
CONFIGURATION
The following options can be added to config.ini

	DISABLEMSSPYNET (Boolean) - Enable or disable the disable_MSSpyNet plugin
		Y	- Enable the disable_MSSpyNet Plugin (DEFAULT).
		N	- Disable the disable_MSSpyNet Plugin.


NOTE
Disabling the plugin does not undo changes made in previous runs.
